Math 241 - Differential Equations

Week-1

The Basics

Let $y$ be a function of $t$. ($y = y(t)$)

Recall that the derivative of $y$ with respect to $t$ is defined as $$y' = \lim_{h \to 0}\frac{y(t+h) - y(t)}{h}$$



Notation We denote the first derivative of $y$ with respect to $t$ by \begin{equation} \label{eq.dy1} \boxed{y' \text{ or }\frac{d y}{d t} } \end{equation}

Similarly, we denote the higher order derivatives of $y$ with respect to $t$ as \begin{equation} \begin{split} y'' \text{ or } \frac{d^2y}{dt^2} &\colon \text{the second derivative (derivative of }y' \text{)}\\ y''' \text{ or } \frac{d^3y}{dt^3} &\colon \text{the third derivative (derivative of }y'' \text{)}\\ y^{(n)} \text{ or } \frac{d^ny}{dt^n} &\colon \text{the }n\text{-th derivative (derivative of }y^{(n-1)} \text{)} \end{split} \end{equation}


When $y = x^3 \colon$ $$y' = 3x^2~,~y'' = 6x~,~y''' = 6~,~ y^{(4)} = 0~,\cdots$$ When $y = e^{2t}$: $$ y' = 2e^{2t}~,~y''=2^2 e^{2t}~,~y''' = 2^3e^{2t}~,~y^{(4)} =2^4e^{2t},\cdots $$ When $y = \sin(t)$: $$ y' = \cos(t)~,~y''=-\sin(t)~,~y''' = -\cos(t)~,~y^{(4)} =\sin(t), \cdots $$


Remark: A function may be dependent on multiple variables instead of one. In this case, the derivatives are taken with respect to each variable, and notation in \eqref{eq.dy1} is slightly changed where $d$'s replaced with $\partial$'s.


Definition: An equation where derivatives are involved is called a Differential Equation (DE).

\begin{equation} \begin{split} \frac{dy}{dt} = y^2-ty + 1 ~ &\text{ or equivalently } y' = y^2-ty + 1,\\ \frac{d^2y}{dx^2} - 2\frac{dy}{dx} + y = e^{x}~&\text{ or equivalently }y'' - 2y' + y = e^{x},\\ \frac{\partial^2 u }{\partial t^2} + \frac{\partial^2 u }{\partial x^2} = 0 ~&\text{ or equivalently }u_{tt} + u_{xx} = 0 \end{split} \end{equation}


Definition: A function which satisfies a differential equation is called a solution.

\begin{equation} \begin{split} y= e^{2t} &\text{ is a solution to the differential equation } y' = 2y\\ y= cos(t) &\text{ is a solution to the differential equation } y'' + y = 0 \\ y= \ln(t) &\text{ is a solution to the differential equation } t\, y' = 1 \end{split} \end{equation} all of which can be obtained through direct substitution. ( Try for yourself!!!)


Definition: If the unknown function in a differential equation depends only one variable then the differential equation is called an Ordinary Differential Equation (ODE), and if the unknown function depends on multiple variables then it is called a Partial Differential Equation (PDE).

**In this course we will work only with the ordinary differential equations**


Definition: The order of a DE is the order of the highest derivative that appears in the equation.

Examples: \begin{equation} \begin{split} y''' + 2e^t y'' + yy' &= 3t^5 ~~~(3\text{rd order ODE}),\\ y^4 - \left( y''\right)^3 + \ln y &= 0 ~~~(2\text{nd order ODE}),\\ ty' - 2y &= \sin t ~~~(1\text{st order ODE}), \\ u_{tt} + u_{xx} &= 0 ~~~(2\text{nd order PDE}), \end{split} \end{equation}


Ordinary Differential Equations



A function $L$ is called linear if $$L(a+b) = L(a) + L(b)$$ for all $a,b$ in the domain of $L$.


Definition: An ODE is called linear if it is linear with respect to all derivatives involved in its equation.


Although mathematically correct, this definition is impractical for identifying whether an ODE is linear. Instead, we use the following identification:

Theorem: An $n$-th order ODE is linear if it is of the form \begin{equation} a_n y^{(n)} + a_{n-1}y^{(n-1)}+\cdots + a_0 y = g \end{equation} where \begin{equation} a_{0},a_1, \cdots,a_n \text{ and } g \text{ are functions of }t \end{equation}

An ODE is called nonlinear if it is not linear.


\begin{equation} \begin{split} t^2y'' - e^ty' + ty = \frac{1}{t}~,&~2\text{nd order, linear ODE},\\ ty'-2y=\sin t ~,&~1\text{st order, linear ODE},\\ (y'')^3 + ty' - y - t^4~&~2\text{nd order, nonlinear ODE},\\ yy' + \frac{1}{t} = 0 ~,&~1\text{st order, nonlinear ODE},\\ y' + \frac{1}{y} + y^2 = t ~,&~1\text{st order, nonlinear ODE},\\ u_{tt} + u_{xx} &= 0 ~~~(2\text{nd order, linear PDE}), \\ u_{t}^2 + u_{xt}^2 &= 0 ~~~(2\text{nd order, nonlinear PDE}), \end{split} \end{equation}


Solve $$y' = 2t$$ We already observe that $t^2$ is a solution (check by direct substitution). We also can work this problem as \begin{align*} \frac{dy}{dt} &= 2t \\ dy &= 2t\, dt\\ \int dy &= \int 2t \, dt \\ y+c_1 &= t^2 + c_2\\ y &= t^2 +C \end{align*} Integrating once generates one constant $C$. Similarly, an $n$-th order ODE requires $n$ integration, each of which generates a new constant. Hence, generally speaking a solution to an $n$-th order ODE is a family of solutions that involves $n$ arbitrary constants.


A family of solutions that involves $n$ arbitrary constants is called a general solution of an $n$-th order linear ODE.

It is possible to impose extra conditions on the function which will ensure that the $n$ constants in the general solution are determined in a unique way. These conditions are called the initial conditions, and an ODE problem with initial conditions are called an initial value problem (IVT)


Find the solution of the initial value problem \begin{equation} \begin{split} y' &= 2t,&~\text{ } \\ y(2)&=3,&~\text{ (initial value)} \end{split} \end{equation}


We already know that $y = t^2+C$ is a general solution. Now, on the one hand \begin{align*} y(2) = 2^2+C = 4+C \end{align*} and on the other hand $y(2)=3$, hence \begin{align*} y(2)= 4+C&=3\\ C &= -1 \end{align*} The solution is $$\boxed{y = t^2 - 1}$$


First Order ODE's


\begin{equation} \label{eqn.yf1} y' = f(t, y) \end{equation}

For arbitrary functions $f$, there is no general method that solves the equation in terms of the elementary functions ( polynomials, exponents, logarithms, trigonometric functions and all other known functions)

We will explore certain strategies which will enable us to solve many kinds of ODE's.


1. Seperable Equations

An equation of the form \eqref{eqn.yf1} is called separable if \begin{equation} f(t,y) = A(t) B(y) \end{equation} for some functions of one variable $A$ and $B$.



Remark: Equation in \eqref{eqn.yf1} is separable when $$ f(t,y) = \frac{a(t)}{b(y)} $$ (This is straightforward when we set $A(t) = a(t) $ and $B(t) = \frac{1}{b(t)}$ in \eqref{eqn.yf1}.)


\begin{equation} \begin{split} \frac{dy}{dt} &= \frac{a(t)}{b(y)}\\ b(y) \, dy &= a(t) \,dt &~\text{ (seperate out $y$ and $t$ terms )} \\ \int b(y) \, dy &= \int a(t) \,dt &~\text{ (integrate both sides) } \end{split} \end{equation} and find each integral separately.


Find the general solution of the ODE: $$y' = \frac{t^2}{1+y^2}$$


Solution: Note that ODE is separable. \begin{equation} \begin{split} \frac{dy}{dt} &= \frac{t^2}{1+y^2}\\ (1+y^2)\, dy&= t^2 \, dt\\ \int(1+y^2)\, dy&= \int t^2 \, dt\\ y+ \frac{y^3}{3} + c_1 &= \frac{t^3}{3} + c_2,\\ y+ \frac{y^3}{3} &= \frac{t^3}{3} + C, \end{split} \end{equation} hence $$\boxed{y+ \frac{y^3}{3} = \frac{t^3}{3} + C}$$ is the general solution. Notice that this solution doesn't have an explicit formula ( i. e. it is not of the form $y =$ a function of $t$ ) but it has an implicit form (where $y$ and $t$ are intertwined in the equation and we can't separate out $y$).


Find the solution of the following IVP \begin{equation} \begin{split} y' &= \frac{x^2}{y(1+x^3)},\\ y(0) &= -1, \end{split} \end{equation}


{Solution} First we find a general solution. Notice that the equation is separable. Hence, \begin{equation} \begin{split} \frac{dy}{dt} &= \frac{x^2}{y(1+x^3)} \\ y \, dy &=\frac{x^2 \, dx}{1+x^3}\\ \int y \, dy &= \int \frac{x^2 \, dx}{1+x^3}\\ \end{split} \end{equation} Use $u$ substitution. Set $u = 1+x^3$, then $du = 3x^2 dx$, or \begin{align*} u &= 1+x^3\\ dx&=\frac{du}{3x^2} \end{align*} Substitution yields \begin{equation} \begin{split} \int y \, dy &=\int \frac{x^2 }{1+x^3}dx\\ &= \int \frac{x^2 }{ u}\frac{du}{ 3x^2} \\ &= \int \frac{1}{3}\frac{du}{u}\\ \frac{y^2}{2} + c_1&=\frac{1}{3}\ln|u| +c_2\\ \frac{y^2}{2} &=\frac{1}{3}\ln\left|1+x^3 \right| + C,\\ y^2&=\frac{2}{3}\ln\left|1+x^3 \right| + D,\\ y&=\pm \sqrt{\frac{2}{3}\ln\left|1+x^3 \right| + D},\ \end{split} \end{equation} hence $y =\pm \sqrt{\frac{2}{3}\ln\left|1+x^3 \right| + D}$ are the general solutions. By setting $t=0$ and using the initial conditions we obtain $y(0) = -1 = \pm\sqrt{ \frac{2}{3}\ln\left|1+x^3 \right| + D } = \pm\sqrt{C}$. Hence the sign must be negative and $D = 1$. The solution of the IVT is: $$\boxed{y = - \sqrt{ \frac{2}{3}\ln\left|1+x^3 \right| + 1 } }$$


2. Homogenous Equations

Consider a Differential Equation of the form $$\frac{dy}{dt} = f(t,y)$$ where $f$ is a function of two variable. The DE is called homogenous equation if $f$ has the form: \begin{equation} \label{eq.f268} f(x,y) = g\left( \frac{y}{t} \right) \end{equation} for some function $g$.


The substitution $$ u= \frac{y}{t} $$ transforms the equation in $\eqref{eq.f268}$ into a separable equation. To see this, observe that $y = ut$ implies \begin{equation} \frac{dy}{dt} = \frac{du}{dt} \cdot t + u\cdot 1 ~{ }\text{ (product rule) } \end{equation} hence \begin{align*} \frac{dy}{dt} &= g\left( \frac{y}{t} \right)\\ \frac{du}{dt} \cdot t + u &= g(u) \\ \frac{du}{dt} \cdot t &= g(u)-u\\ \frac{du}{dt} &= \frac{g(u)-u}{t} ~\text{ (separable equation)} \end{align*} and we solve the separable equation.


Find the general solution of the differential equation $$ y' = \frac{x^2+xy+y^2}{x^2} $$


\begin{align*} y' &= \frac{x^2+xy+y^2}{x^2} \\ &= \frac{x^2}{x^2} + \frac{xy}{x^2} + \frac{y^2}{x^2} \\ &=1+\frac{y}{x} + \left(\frac{y}{x}\right)^2 ~\text{ (homogenous equation)} \end{align*} Let $$u = \frac{y}{t}$$ then \begin{align*} y &= ut\\ \frac{dy}{dt}&=\frac{du}{dt} \cdot t + u \end{align*} Hence \begin{align*} \frac{dy}{dt} &=1+\frac{y}{x} + \left(\frac{y}{x}\right)^2\\ \frac{du}{dt} \cdot t + u&=1+u+u^2\\ \frac{du}{dt} \cdot t&=1+u^2\\ \frac{du}{dt} &=\frac{(1+u^2)}{t}~\text{ (separable equation)}\\ \frac{du}{(1+u^2)} &= \frac{dt}{t}\\ \int \frac{du}{(1+u^2)} &=\int \frac{dt}{t}\\ \arctan{u}&= \ln|t| + C\\ \arctan{\left(\frac{y}{x}\right)}&= \ln|t| + C\\ \frac{y}{x} &= \tan{\left( \ln |t| + C \right)}\\ y&= x\, \tan{\left( \ln |t| + C \right)} \end{align*} and the answer is $$\boxed{ y = x\, \tan{\left( \ln |t| + C \right)} }$$


Find the solution of the I. V. P. \begin{align*} \frac{dy}{dt} &= \frac{t+3y}{t-y},\\ y(1) &=2, \end{align*}


\begin{equation} \label{eq.dy479} \begin{split} \frac{dy}{dt} &= \frac{t+3y}{t-y} \\ &= \frac{t\left(1+3\frac{y}{t} \right)}{t\left(1-\frac{y}{t}\right)} \\ &= \frac{1+3\frac{y}{t} }{1-\frac{y}{t}} ~\text{ (homogenous equation)} \end{split} \end{equation} Let \begin{equation} \label{eq.u488} u = \frac{y}{t} \end{equation} then \begin{equation} \label{equ.y494} \begin{split} y &= ut\\ \frac{dy}{dt}&=\frac{du}{dt} \cdot t + u \end{split} \end{equation} Use \eqref{eq.dy479}, \eqref{eq.u488} and \eqref{equ.y494} to obtain \begin{equation} \label{eq.dut502} \begin{split} \frac{du}{dt} \cdot t + u&= \frac{1+3u }{1-u} \\ \frac{du}{dt} \cdot t &= \frac{1+3u }{1-u}-u\\ &= \frac{1+3u-u+u^2 }{1-u}\\ &= \frac{1+2u+u^2 }{1-u}\\ &= \frac{(1+u)^2 }{1-u}\\ \frac{du}{dt} &= \frac{(1+u)^2 }{t(1-u)}~\text{ (separable equation)}\\ \frac{{(1-u)} }{(1+u)^2} du &= \frac{1}{t}dt\\ \int \frac{{(1-u)} }{(1+u)^2} du &= \int \frac{1}{t}dt\\ \end{split} \end{equation} For the right hand side integral we use the substitution \begin{align*} w &= 1+u\\ dw&=du \end{align*} and obtain \begin{equation} \label{eq.if519} \begin{split} \int \frac{{(1-u)} }{(1+u)^2} du &= \int \frac{{(1-(w-1))} }{w^2} dw\\ &= \int \frac{2-w}{w^2} dw \\ &= \int \left( \frac{2}{w^2} - \frac{w}{w^2} \right) dw\\ &= 2\int \left(w^{-2} - w^{-1} \right)dw\\ &=- 2w^{-1} - \ln |w| + C\\ &=-\frac{2}{(1+u)} -\ln|1+u| + c_1 \end{split} \end{equation} Since $\int \frac{1}{t}dt = \ln|t| +c_2$, \eqref{eq.dut502} and \eqref{eq.if519} yields \begin{equation} \label{eq.u536} \begin{split} \frac{2}{(1+u)} -\ln|1+u| &= \ln|t| +C \\ \frac{2}{1+\frac{y}{t}} -\ln\left|1+\frac{y}{t} \right| &= \ln|t| +C \end{split} \end{equation} We use initial conditions to find $C$. In this case $t=1$, $y=2$ and when substituted in \eqref{eq.u536} we get \begin{align*} \frac{2}{1+2} - \ln|1+2| &= \ln{1} + C\\ \frac{2}{3} - \ln|3|&=C \end{align*} and the final answer is $$\boxed{ \frac{2}{1+\frac{y}{t}} -\ln\left|1+\frac{y}{t} \right| = \ln|t| +\frac{2}{3} - \ln|3| }$$


Find the solution of the initial value problem \begin{align*} \frac{dy}{dt} &= -y^2\cos{t}, \\ y(0)&= \frac{1}{2} \end{align*}


\begin{equation} \begin{split} \frac{dy}{dt} &= -y^2\cos{t} ~~\text{ (separable equation)}\\ \frac{-1}{y^2}\,dy&= \cos{t}\, dt \\ \int \frac{-1}{y^2}\,dy&= \int \cos{t}\, dt \\ \frac{1}{y} &= \sin{t} + C\\ y&= \frac{1}{\sin{t} + C} \end{split} \end{equation} when initial value condition is used we get $$y(0) = \frac{1}{2} = \frac{1}{0+C} = \frac{1}{C}$$ hence $C=2$, and the solution is $$\boxed{y = \frac{1}{\sin{t} + 2}}$$


Find the general solution of the differential equation $$ \frac{dx}{dt} = \frac{(t+1)x}{(x^2+1)(t^2-4)} $$


\begin{equation} \begin{split} \frac{dx}{dt} &= \frac{(t+1)x}{(x^2+1)(t^2-4)} \\ &= \frac{t+1}{t^2-4}\cdot \frac{x}{x^2+1}~~\text{ (separable equation)}\\ \frac{x^2+1}{x}\,dx &= \frac{t+1}{t^2-4} \, dt\\ \int \frac{x^2+1}{x}\,dx &= \int \frac{t+1}{t^2-4} \, dt \end{split} \end{equation} Let's calculate the integrals on each side. The left hand side: \begin{align*} \int \frac{x^2+1}{x}\,dx &= \int \left(\frac{x^2}{x} + \frac{1}{x}\right)\,dx \\ &=\int\left( x + \frac{1}{x} \right)dx\\ &= \frac{x^2}{2} + \ln|x| + c_1 \end{align*} and the right hand side: \begin{align*} \int \frac{t+1}{t^2-4} \, dt &= \int \frac{t+1}{(t-2)(t+2)} \, dt \\ &\left( \begin{array}{rl} \frac{t+1}{(t-2)(t+2)} &= \frac{A}{t-2} + \frac{B}{t+2} \\ A(t+2) + B(t-2) &= t+1 \\ t=2\colon 4A=3 &\to A=\frac{3}{4}\\ t=-2\colon -4B = -1 &\to B=\frac{1}{4} \end{array} \right) \\ &=\int\left( \frac{3}{4}\,\frac{1}{t-2} + \frac{1}{4}\,\frac{1}{t+2} \right)dt \\ &= \frac{3}{4}\ln|t-2| + \frac{1}{4}\ln|t+2| + c_2 \end{align*} and when we combine the results we obtain (on combining the constants on the left hand side) $$\boxed{\frac{x^2}{2} + \ln|x| = \frac{3}{4}\ln|t-2| + \frac{1}{4}\ln|t+2| + C }$$


Find the general solution of the differential equation $$x y' - y = y(\ln{y} - \ln{x})$$


\begin{equation} \label{eq.xy643} \begin{split} x \frac{dy}{dx} &= y(\ln{y} - \ln{x}) + y \\ x \frac{dy}{dx} &= y \ln{ \frac{y}{x}} + y \\ \frac{dy}{dx} &=\frac{ y}{x} \ln{ \frac{y}{x}} + \frac{y}{x} ~~\text{ (homogenous equation)}\\ \end{split} \end{equation} Let \begin{equation} \label{eq.u653} u = \frac{y}{x} \end{equation} then \begin{equation} \label{eq.y658} \begin{split} y &= ux\\ \frac{dy}{dt}&=\frac{du}{dx} \cdot x + u \end{split} \end{equation} Use \eqref{eq.xy643}, \eqref{eq.u653} and \eqref{eq.y658} to obtain \begin{equation} \begin{split} \frac{dy}{dx} &=\frac{ y}{x} \ln{ \frac{y}{x}} + \frac{y}{x}\\ \frac{du}{dx} \cdot x + u&= u\ln{u} + u\\ \frac{du}{dx} &=\frac{1}{x}u\ln{u} ~~\text{ (separable equaiton)}\\ \frac{1}{u \ln u} du &= \frac{1}{x}dx\\ \int \frac{1}{u \ln u} du &= \int \frac{1}{x}dx ~~~ \left\{ \begin{array}{rl} w&=\ln u \\ dw &= \frac{1}{u} du\\ du&=u\, dw \end{array} \to \frac{1}{u \ln u} du = \frac{1}{u w} u \, dw = \frac{1}{w} dw \right. \\ \int \frac{1}{w} dw &= \int \frac{1}{x}dx\\ \ln|w| &= \ln|x| + C\\ \ln| {\ln |u|} | &=\ln|x| + C\\ \ln \left| \ln \left| \frac{y}{x} \right| \right| &= \ln|x| + C \end{split} \end{equation} hence the answer is $$\boxed{ \ln \left| \ln \left| \frac{y}{x} \right| \right| = \ln|x| + C }$$


Week-2

Definition.

Any first order linear differential equation has the form \begin{equation} \label{eq.aby} a(t)\cdot y' + b(t)\cdot y = c(t) \end{equation}

By dividing each side by $a(t)$, \eqref{eq.aby} can be rewritten as \begin{equation} \label{eq.ypg1} y' + p(t)\cdot y = g(t) \end{equation} where $p(t) = \frac{b(t)}{a(t)}$ and $q(t) = \frac{c(t)}{a(t)}$. $$ \begin{array}{l} \text{(i) the coefficient of }y' \text{ is }1 \\\text{(ii) } y' \text{ and }y \text{ terms are on the same side of the equation} \\ \text{(iii) the rest is on the other side} \end{array} $$


Idea.

Observe that the product rule implies \begin{equation} \label{eq.prodrl1} ( \mu \cdot y)' = \mu \cdot y' + \mu' \cdot y \end{equation} for a function $\mu(t)$. We want to choose a suitable $\mu$ such that when when we multiply the left hand side of \eqref{eq.ypg1} by $\mu$ we obtain \eqref{eq.prodrl1}. This works if we find a $\mu$ such that \begin{equation} \begin{split} \mu\cdot y' + \mu\cdot p\cdot y &= \mu \cdot y' + \mu' \cdot y\\ \mu \cdot p&= \mu' \\ \mu \cdot p&= \frac{d\mu}{dt}~~\text{ (separable eq.)}\\ p\, dt &= \frac{1}{\mu} d\mu\\ \int p\, dt &=\int \frac{1}{\mu} d\mu\\ \int p\, dt &= \ln{\mu}\\ \mu &= e^{\int p\, dt } \end{split} \end{equation} Hence, \begin{equation} \label{eq.mu532} \boxed{\mu = e^{\int p\, dt } } \end{equation} works.


Strategy.

The strategy is to multiply each side of \eqref{eq.ypg1} by $\mu = e^{\int p\, dt }$ and proceed. When we do that we obtain \begin{equation} \begin{split} \mu \, y' +\mu\, p\, y &= \mu \, g\\ e^{\int p\, dt } \, y' + e^{\int p\, dt } p\, y &= e^{\int p\, dt } \, g\\ \left( e^{\int p\, dt } \, y \right)' &= e^{\int p\, dt } \, g ~~~\text{ ( from }(e^{U})' = e^{U}\,U' \text{ rule )} \\ \int \left( e^{\int p\, dt } \, y \right)'\,dt &= \int e^{\int p\, dt } \, g \, dt\\ e^{\int p\, dt } \, y &= \int e^{\int p\, dt } \, g \, dt \\ y&= \frac{1}{e^{\int p\, dt }} \int e^{\int p\, dt } \, g \, dt \end{split} \end{equation} or equivalently \begin{equation} \label{eq.ymu1} \boxed{ y = \frac{1}{\mu} \int \mu \, g \, dt } \end{equation}


Remark.

You don't need to memorize the formula in \eqref{eq.ymu1}.You can proceed by multiplying each side of \eqref{eq.ypg1} by $\mu$ after you obtain $\mu$ from \eqref{eq.mu532}.


Find the general solution of the differential equation $$ y' - 2y = 3e^{5t} $$

Note that this is a linear equation with $$ p(t) = -2$$ and \begin{align*} \mu &= e^{\int p\, dt } \\ &=e^{\int -2\, dt }\\ &=e^{-2t + C}\\ &= e^{-2t} ~~~\text{ ( choose }C=0 \text{ as we have freedom)} \end{align*} Multiply both sides of the differential equation by $\mu = e^{-2t}$ to obtain \begin{align*} e^{-2t}\,y' - e^{-2t}\,2y &= e^{-2t}\, 3e^{5t} \\ \left( e^{-2t}\,y \right)' &= 3e^{3t} \\ e^{-2t}\,y &= \int 3e^{3t} \, dt \\ e^{-2t}\,y&= e^{3t} + C\\ y &= e^{2t}\left(e^{3t} + C\right) \end{align*} hence the general solution is $$\boxed{ y =e^{5t} + Ce^{2t} }$$



Find the solution of the I. V. P. \begin{align*} t\, y' + 3\,y &= \ln{t},\\ y(1)&=-1, \end{align*}

Divide the equation by $t$ and obtain \begin{equation} \label{eq.tyyp1} \begin{split} t\, y' + 3\,y &= \ln{t}, ~~~\text{ (divide by the coefficient of }y' \text{ )}\\ y' + \frac{3}{t}\,y&=\frac{\ln{t}}{t} \end{split} \end{equation} Then \begin{align*} \mu &= e^{\int \frac{3}{t}\, dt }\\ &= e^{3\ln{t}} ~~~\text{ ( we choose }C=1 \text{ )}\\ &=\left(e^{\ln{t}}\right)^3\\ &=t^{3} \end{align*} Multiply each side of \eqref{eq.tyyp1} by $\mu = t^3$ to obtain \begin{align*} t^3\,y' + 3t^2\,y=&t^2\ln{t}\\ \left(t^3 y \right) =& t^2\ln{t}\\ t^3 y =& \int t^2\ln{t} \,dt ~~\left( \begin{array}{l}\text{Integration by Parts :} \\ \boxed{\int U\,dV = UV-\int V\,dU} \\ U=\ln{t}~,~dV= t^2 \, dt \\ dU= \frac{1}{t}\, dt~,~V =\frac{t^3}{3} \end{array} \right)\\ =& \frac{t^2}{3}\, \ln{t} - \int \frac{t^2}{3} dt\\ t^3 y=& \frac{t^2}{3}\, \ln{t} -\frac{t^3}{9} + C\\ y &= \frac{\ln{t}}{3t} - \frac{1}{9} + \frac{C}{t^3} \end{align*} and when we use the initial condition in the last equation we obtain \begin{align*} y(1) &= -1 =-\frac{1}{9} + C\\ C&= -\frac{8}{9} \end{align*} and the solution of IVT is $$\boxed{y = \frac{\ln{t}}{3t} - \frac{1}{9} - \frac{8}{9t^3} }$$



Find the solution of the D. E. $$\sin{t} \, y' = \tan{t} - \cos{t}\,y $$

\begin{equation} \label{eq.ex3} \begin{split} \sin{t} \cdot y' &= \tan{t} - \cos{t}\cdot y\\ \sin{t} \cdot y' + \cos{t} \cdot y &= \tan{t} \\ y' + \frac{\cos{t}}{\sin{t}} \cdot y &= \frac{\tan{t}}{\sin{t}}\\ y' + \frac{\cos{t}}{\sin{t}}\, y &= \frac{1}{\cos{t}} \end{split} \end{equation} and \begin{equation} \label{eq.mu3} \begin{split} \mu &= e^{\int \frac{\cos{t}}{\sin{t}}\, dt }\\ &=e^{\ln \sin{t}}\\ &=\sin{t} \end{split} \end{equation} Multiply \eqref{eq.ex3} by $\mu = \sin{t}$ to obtain \begin{align*} \sin{t} \cdot y' +\cos{t} \cdot y &= \frac{\sin{t}}{\cos{t}}\\ (\sin{t} \cdot y)' &= \frac{\sin{t}}{\cos{t}}\\ \sin{t} \cdot y &= \int \frac{\sin{t}}{\cos{t}}\, dt\\ \sin{t} \cdot y &=-\ln |\cos{t}| + C \end{align*} hence the general solution is $$\boxed{ y = \frac{-\ln|\cos{t}| + C}{\sin{t}} }$$




Definition.

A Differential Equation of the form \begin{equation} \label{eq.bern} \boxed{y' + p(t)\cdot y = q(t)\cdot y^{b}} ~~\text{ where }b\text{ is a real number} \end{equation} is called a Bernoulli Equation.

Note that when $b=0$ or $b=1$ the equation is linear: \begin{align*} y' + p(t)\cdot y = q(t)&~~~\text{ (}b=0\text{)}\\ y' + p(t)\cdot y = q(t)\cdot y&~~~\text{ (}b=1\text{)} \end{align*}


Strategy.

When $b\neq 0$ or $b\neq 1$, the substitution \begin{equation} u=y^{1-b} \end{equation} transforms Bernoulli Equation into a linear differential equation. To see this, notice that \begin{equation} \label{eq.bernu1} u' = (1-b)y^{-b} y' \end{equation} Multiply \eqref{eq.bern} by $ (1-b)y^{-b}$ and use \eqref{eq.bernu1} to obtain \begin{align*} (1-b)y^{-b}y' + (1-b)y^{-b} p(t)\cdot y &= (1-b)y^{-b}q(t)\cdot y^{b}\\ u' + (1-b) p(t) \cdot u &= (1-b) q(t) ~~~\text{ (linear first order ODE)} \end{align*}


Find the solution of the I. V. P. \begin{align*} y' - \frac{y}{2t} &= t^2\, y^3,\\ y(2)&=1, \end{align*}

This is a Bernoulli equation with $b = 3$. We set $$ u = y^{1-3} = y^{-2} $$ Then $$u' = -2y^{-3}y'$$ Multiply each side of D. E. by $-2y^{-3}$ to obtain \begin{align*} -2y^{-3}y' + 2y^{-3}\frac{y}{2t} &=-2y^{-3} t^2\, y^3\\ -2y^{-3}y' + \frac{1}{t}y^{-2} &= -2t^2\\ u' + \frac{1}{t} u &= -2t^2 ~~~\left ( \begin{array}{l} \text{first order linear D. E.} \\ \mu = e^{\int \frac{1}{t}dt}=e^{\ln{t}}=t \end{array} \right)\\ tu' + u &= -2t^3\\ (tu)'&=-2t^3\\ tu &= \int -2t^3 dt\\ tu&= -\frac{t^4}{2} + C\\ u&=-\frac{t^3}{2} + \frac{C}{t}~~~ \text{ (general solution)} \end{align*} Use the initial condition in the last result to obtain \begin{align*} y(2)&=1=-4 + \frac{C}{2}\\ C&=10 \end{align*} hence $$ u = y^{-2} = -\frac{t^3}{2} + \frac{10}{t} $$ and the answer is $$\boxed{ y = \left( -\frac{t^3}{2} + \frac{10}{t} \right)^{-2} }$$



Find the general solution of the D. E. $$ y\left( 1-x-5y^{-3} \right)\,dx + 3x \,dy = 0 $$

\begin{equation} \begin{split} y\left( 1-x-5y^{-3} \right)\,dx + 3x \,dy &= 0 ~~~\text{ (divide by } dx \text{)}\\ y\left( 1-x-5y^{-3} \right) + 3x \,\frac{dy}{dx} &= 0 \\ 3x \,\frac{dy}{dx} + y( 1-x) &= 5y^{-2} \\ y' + y\frac{( 1-x)}{3x} &= \frac{5}{3x} y^{-2} ~~~ \left( \begin{array}{l} \text{Bernoulli equation with }b=-2 \\ u=y^{1-(-2)} = y^{3}\\ u'=3y^2\,y' \end{array} \right)\\ 3y^2\,y' + 3y^2\,y\frac{( 1-x)}{3x} &= 3y^2\,\frac{5}{3x} y^{-2}\\ u' +\frac{1-x}{x}\,u &=\frac{5}{x} ~~~\left ( \begin{array}{l} \text{linear D. E.} \\ \mu = e^{\int \frac{1-x}{x}dx}\\=e^{\int \frac{1}{x} - 1 \,dx} \\ = e^{\ln x - x} \\= x\,e^{-x} \end{array} \right)\\ x\,e^{-x}\left( u' +\frac{1-x}{x}\,u \right)&=x\,e^{-x}\left(\frac{5}{x} \right)\\ x\,e^{-x} u' + (1-x)e^{-x} u &= 5e^{-x}\\ \left( x e^{-x} u \right)' &= 5e^{-x}\\ x e^{-x} u &= \int 5e^{-x}\,dx\\ x e^{-x} u&=-5e^{-x} + C\\ u&=\frac{-5 + Ce^{x}}{x}\\ y^3&=\frac{-5 + Ce^{x}}{x} \end{split} \end{equation} hence the general solution is $$\boxed{ y = \left( \frac{-5 + Ce^{x}}{x} \right)^{\frac{1}{3}} }$$
